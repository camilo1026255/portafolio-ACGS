MI PORTAFOLIO

Python y PHP; desarrollo web utilizando HTML, CSS; y
manejo bases de datos como MySQL. Además, estoy
familiarizado con metodologías ágiles como Scrum y
Kanban, y tengo experiencia en el uso de Git para el control

destaco por mi capacidad para trabajar en equipo,
comunicarme de manera efectiva con audiencias técnicas
y no técnicas, adaptarme rápidamente a nuevas tecnologías y
mantener un enfoque en la entrega de proyectos de alta
calidad dentro de los plazos establecidos.
